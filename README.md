## 주요 기능

- [x] firebase를 사용하여 google 로그인 구현
- [x] cloudinary를 활용한 상품 등록 기능
  - [x] admin 유저일 경우만 등록이 가능하다
- [x] 장바구니 기능
  - [x] 상품 추가
  - [x] 상품 삭제
  - [x] 상품 수량 변경
  - [x] 선택된 상품 총 금액 보여주기
- [x] react-query를 활용한 데이터 패칭 및 업데이트
